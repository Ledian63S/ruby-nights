# Changelog

## [1.0.5] - 2026-03-08
### Changed
- Renamed extension and all themes from Lumos to Ruby Nights

## [1.0.0] - 2026-03-08
### Changed
- Complete redesign — replaced previous variants (Ash, Dark, Onyx) with four new ruby/garnet variants
- New variants: Ruby, Ruby Midnight, Garnet, Garnet Midnight
- Dark ruby and garnet background tones replace previous neutral/blurple palette
- Darker sidebar, lighter editor for clear visual separation
- Midnight variants feature hard black borders between UI regions for crisp definition
- Updated breadcrumb, active tab, selection, bracket match, and scrollbar colors to match palette
- Gold accent cursor, muted sage green for git modified files
- Improved active line highlight visibility across all variants
- Light rose cursor color for better visibility against dark backgrounds
